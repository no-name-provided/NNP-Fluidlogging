This mod adds very general fluidlogging to Minecraft+NeoForge. As it is mixin based, it can be adapted for other
mod loaders with relative ease. I will not be doing this. As the vanilla fluid system is seldom meaningfully updated,
it should be easy to port this forward. I intend to do that, once this mod reaches beta status. I will not backport,
as older versions have alternatives that are easily available, and claim to support general fluidlogging.

I don't intend to push this mod past late beta or early release. I am providing it because I think it's crazy no one
else has bothered. It is my hope other, more interested, folks will contribute to this foundation and build this up into
a great mod. If not, oh well, this will still be pretty decent.

Anyway, feel free to request features, and please do report bugs. However, if you aren't afraid to try
your hand at coding, consider taking a crack at fixing some of those bugs, or adding whatever feature
you want to see. PRs appreciated; just make sure you agree to the license.
